Welcome to the official repository for the **MenaML (Middle East and North Africa Machine Learning) Winter School** hands-on lab sessions. 

The MenaML Winter School is dedicated to empowering the next generation of AI leaders in the MENA region. This repository hosts the code, notebooks, and resources used during our practical sessions, designed to provide experience with the latest machine learning tools and techniques.
